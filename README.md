# Cannae

> *Outthink larger forces through superior preparation and memory.*

AI-powered business operating system. An Agent Council (CEO, Coach, SEO, CFO) with persistent graph memory that connects your decisions across time.

## Quick Start

```bash
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env

python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt

python app.py
# → http://localhost:8000
```
